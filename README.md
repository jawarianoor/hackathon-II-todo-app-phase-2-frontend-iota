# Todo App – Phase II

**Full-Stack Web Application with Authentication**

> Phase II of **“The Evolution of Todo: From CLI to Distributed Cloud-Native AI Systems”**

## Overview

This phase evolves the project from a single-user CLI todo app into a secure, multi-user full-stack web application with authentication, persistent storage, and a modern React UI.

## Features

- User Authentication (Better Auth)
- Per-user task isolation
- Add, update, view, delete tasks
- Toggle task completion
- Secure JWT-based API

## Tech Stack

Frontend: Next.js 16+, TypeScript (strict), Tailwind CSS  
Backend: FastAPI, Python 3.11, SQLModel (async)  
Database: Neon PostgreSQL
